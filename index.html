profil jurusan

Program Pengembang perangkat lunak dan gim atau sebelumnya keahlian rekayasa perangkat lunak (rpl)adalah salah satu kompetensi keahlian dalam bidang teknologi komputer dan informatika yang secara khusus mempelajari tentang pemrograman komputer. lulusan RPL dimasa ini sangat dibutuhkan untuk menjawab banyak nya kebutuhan industri bagi ketersedian tenaga teknisi dalam bidang Pengembang software. beberapa ruang lingkup perkerjaan di bidang RPL adala software developer,game development,software tester,softrware engineering,software analis dan integrator,konsultan it dan programer.

Peluang karir jurusan rpl bisa melingkup software engineer/programmer, mobile computing developer,IT consultant, System analyst, game developer,software tester dan masih banyak lagi prospek karir lainnya.
